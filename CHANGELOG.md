# Kola UI – Changelog

## v1.0.0 (2026-03-29)

- Initial public release
- Component-based UI framework (BaseComp, BaseWindow, Div, etc.)
- Advanced text input (selection, clipboard, undo/redo)
- SVG and image support (SVG from string, image URL/alt)
- Real-time rendering (Snake game demo, 30+ FPS)
- ResizableDiv (mouse-resizable panel)
- Tailwind-style string parser (bg, rounded, w/h, grid, flex, gap, etc.)
- Modal dialogs, menu bar, segmented select, scroll view, etc.
- Custom layout engines (Flex, Grid, Block, Absolute)
